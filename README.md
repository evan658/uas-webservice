Project UAS mata kuliah Pemrograman Web Service  
Menggunakan Laravel dan MySQL

## Fitur

- CRUD Data Lagu
- Database Migration

## Teknologi
- Laravel
- PHP
- MySQL
- Blade Template

## Cara Menjalankan
1. Clone repository
2. Jalankan `composer install`
3. Copy `.env.example` menjadi `.env`
4. Setting database
5. Jalankan `php artisan migrate`
6. Jalankan `php artisan serve`
